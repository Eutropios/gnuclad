5.3 Adding configuration options

When adding a configuration option, be sure to implement it at least in the following places:

  • gnuclad.h: members of the Cladogram class (declaration)
  • gnuclad.cpp: in the constructor of the Cladogram object (initialisation)
  • gnuclad.cpp: in the parseOptions function
  • generatorCONF.cpp: insert description of the new option
  • the texinfo documentation, chapter Config Options: insert the new config file created by generatorCONF
  • third party gnuclad configuration files you have access to

In order to work with the most current config options, open the 'gnuclad.h' source file and search for CONFIG OPTIONS. This will take you straight to the declarations.

5.4 Adding a parser

Parsers read data from various sources into the Cladogram object. For detailed information, see How to write a parser.

parserXXX.h: 

     #include "gnuclad.h"
     derive class from Parser:
       class ParserXXX: public Parser
     declare constructor and destructor
     declare public method:
       void parseData(Cladogram * clad, std::ifstream * fp);

parserXXX.cpp: 

     #include "parserXXX.h"
     implement constructor and destructor
     implement parseData(Cladogram * clad, std::ifstream * fp);

gnuclad.cpp: 

     #include "parserXXX.h"
     update inFormats
     add choser:
       else if(inputExt == "XXX") parser = new ParserXXX;

5.5 Adding a generator

Generators write data from the Cladogram object to a specified output format. For detailed information, see How to write a generator.

generatorXXX.h: 

     #include "gnuclad.h"
     derive class from generator:
       class GeneratorXXX: public Generator
     declare constructor and destructor
     declare public method:
       void writeData(Cladogram * clad, std::ofstream * fp);

generatorXXX.cpp: 

     #include "generatorXXX.h"
     implement constructor and destructor
     implement writeData(Cladogram * clad, std::ofstream * fp);

gnuclad.cpp 

     #include "generatorXXX.h"
     update outFormats
     add choser:
       else if(outputExt == "XXX") generator = new GeneratorXXX;
